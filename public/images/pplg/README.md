# PPLG Activity Images

This directory contains images for PPLG (Pengembangan Perangkat Lunak dan Gim) activities and lab documentation.

## Required Images

The following images are referenced in the PPLG page and need to be added:

### Lab Activities
- `lab-coding.jpg` - Students learning basic programming in the lab
- `web-development.jpg` - Web development practice sessions
- `mobile-app.jpg` - Mobile app development activities
- `game-dev.jpg` - Game development workshops
- `database-lab.jpg` - Database management practice
- `team-project.jpg` - Team software development projects
- `coding-competition.jpg` - Programming competitions between students
- `software-testing.jpg` - Software testing and debugging sessions

## Image Specifications
- **Format**: JPG or PNG
- **Recommended Size**: 400x300 pixels or larger
- **Aspect Ratio**: 4:3 or 16:9
- **Quality**: High quality, clear images showing students actively engaged in PPLG activities

## Usage
These images are displayed in the PPLG page gallery section to showcase:
- Laboratory activities
- Student projects
- Learning environments
- Practical sessions
- Competitions and events

## Notes
- Ensure all images have proper alt text for accessibility
- Images should represent actual PPLG activities at SMKN 4 Bogor
- Maintain consistent image quality and style across all photos
