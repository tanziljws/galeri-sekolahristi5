# Galeri Sekolah Umum

Direktori ini berisi foto-foto untuk galeri sekolah umum yang akan ditampilkan di halaman utama "Galeri Agenda Sekolah".

## Kategori Foto

### 1. **Kegiatan Sekolah** (`kegiatan_sekolah`)
- Upacara bendera
- Rapat guru
- Kegiatan administrasi
- Kegiatan umum sekolah

### 2. **Acara Khusus** (`acara`)
- Hari besar nasional
- Acara perpisahan
- Wisuda
- Seminar sekolah

### 3. **Prestasi Siswa** (`prestasi`)
- Lomba antar sekolah
- Penghargaan siswa
- Kompetisi akademik
- Prestasi non-akademik

### 4. **Fasilitas Sekolah** (`fasilitas`)
- Gedung sekolah
- Ruang kelas
- Laboratorium
- Perpustakaan
- Kantin
- Lapangan olahraga

### 5. **Kegiatan Umum** (`kegiatan_umum`)
- Kegiatan ekstrakurikuler
- Kegiatan sosial
- Kunjungan tamu
- Kegiatan lainnya

### 6. **Lainnya** (`lainnya`)
- Foto yang tidak masuk kategori di atas

## Spesifikasi Foto

- **Format**: JPG, PNG, GIF
- **Ukuran Maksimal**: 2MB
- **Resolusi**: Minimal 400x300 pixel
- **Kualitas**: Foto yang jelas dan tidak blur

## Penggunaan

Foto-foto ini akan ditampilkan di:
- Halaman utama website (section "Galeri Agenda Sekolah")
- **TIDAK AKAN** muncul di halaman jurusan (PPLG, TJKT, TPFL, TO)

## Manajemen

Admin dapat mengelola foto-foto ini melalui:
- **URL**: `/admin/school-galleries`
- **Fitur**: Tambah, edit, hapus, aktifkan/nonaktifkan

## Pemisahan Sistem

⚠️ **PENTING**: Sistem galeri ini terpisah dari sistem kegiatan jurusan
- **Galeri Sekolah** → Halaman utama galeri
- **Kegiatan Jurusan** → Halaman jurusan masing-masing

Kedua sistem tidak akan tercampur!
