# Foto Fasilitas SMK Negeri 4 Kota Bogor

## Cara Menambahkan Foto Fasilitas

Untuk menampilkan foto-foto fasilitas sekolah yang Anda kirim, silakan ikuti langkah berikut:

### 1. Simpan Foto ke Folder Ini
Ganti file-file placeholder berikut dengan foto asli yang Anda kirim:

- `gedung-utama.JPG` - Foto gedung utama sekolah
- `kompleks-sekolah.JPG` - Foto kompleks sekolah dari udara
- `gedung-rpl.png` - Foto gedung RPL dengan spanduk "TEKNIK KOMPUTER DAN JARINGAN"
- `gedung-smkn4.JPG` - Foto gedung SMKN 4 dengan tulisan "SMKN 4 BOGOR"

### 2. Format Foto
- **Format**: JPG atau PNG
- **Ukuran**: Maksimal 2MB per foto
- **Resolusi**: Minimal 800x600 pixel
- **Nama file**: Gunakan nama yang sudah disediakan

### 3. Cara Upload
1. Buka folder `public/images/fasilitas/`
2. Ganti file placeholder dengan foto asli
3. Pastikan nama file sama persis dengan yang ada di kode
4. Refresh halaman fasilitas untuk melihat hasilnya

### 4. Foto yang Diperlukan
Berdasarkan foto yang Anda kirim, silakan simpan dengan nama:

1. **gedung-utama.JPG** - Foto gedung dengan atap merah dan dinding biru
2. **kompleks-sekolah.JPG** - Foto kompleks sekolah dari udara dengan banyak pohon
3. **gedung-rpl.png** - Foto gedung dengan spanduk "TEKNIK KOMPUTER DAN JARINGAN"
4. **gedung-smkn4.JPG** - Foto gedung dengan tulisan "SMKN 4 BOGOR"

### 5. Tips
- Pastikan foto berkualitas baik dan tidak blur
- Foto akan otomatis di-resize oleh CSS
- Hover pada foto untuk melihat informasi detail
- Layout akan otomatis responsive di mobile

Setelah foto diupload, halaman fasilitas akan menampilkan galeri foto yang menarik dengan efek hover dan overlay informasi.
